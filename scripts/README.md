# Scripts

This folder contains utility scripts for data processing, visualization generation, and analysis.

## Visualization Scripts
- **generate_visualizations.py** - Main visualization generation script
- **generate_data_figures.py** - Generates data distribution and statistics figures
- **generate_interpretability_images.py** - Creates model interpretability visualizations
- **visualize_adnimerge_usage.py** - Visualizes ADNIMERGE dataset usage patterns

## Data Processing Scripts
- **extract_adni_samples.py** - Extracts and processes ADNI dataset samples
- **generate_adni_json.py** - Generates JSON metadata for ADNI data

## Analysis Scripts
- **check_adnimerge_usage.py** - Analyzes ADNIMERGE dataset usage
- **quick_adni_check.py** - Quick validation check for ADNI data

## Usage

All scripts should be run from the project root directory:

```bash
python scripts/script_name.py
```

Make sure all dependencies from `requirements.txt` are installed before running any scripts.
